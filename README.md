# 🪖 Smart Helmet App

> A next-generation smart helmet ecosystem combining **IoT, safety, and seamless connectivity** — powered by Flutter and ESP32.

---

## 🚀 Overview

The **Smart Helmet App** enhances rider safety and experience with:

- 📡 Real-time communication with helmet hardware  
- 🎧 Audio control & call handling  
- ⚠️ Safety alerts & monitoring  
- 🔋 Battery status tracking  
- 🧭 Smart navigation integration *(planned)*  

Built alongside a custom helmet powered by **ESP32 & ESP32-S3**.

---

## 🧠 Key Features

### 🔗 Helmet Connectivity
- Bluetooth Classic + BLE  
- Seamless pairing  
- Real-time data exchange  

### 🎵 Audio & Calls
- Music playback control  
- Call handling  
- DSP-based audio processing  

### ⚡ Battery Monitoring
- Live battery percentage (SoC)  
- Usage prediction *(planned)*  

### 🛡️ Safety System
- Smart alerts  
- Rider assistance *(in progress)*  

---

## 📱 UI Highlights

- 🎬 Scroll-driven onboarding animations  
- 🌑 Fully dark-themed UI  
- ⚡ Smooth interactions  
- 🎯 Clean, minimal design  

---

## 🏗️ Tech Stack

- **Frontend:** Flutter  
- **Communication:** Bluetooth Classic + BLE  
- **MCU:** ESP32 + ESP32-S3  
- **Audio Processing:** DSP  
- **Architecture:** Modular Flutter

## ⚙️ Getting Started

### Prerequisites
- Flutter SDK  
- Android Studio / VS Code  
- Physical Android device (recommended)

### Installation

```bash
git clone https://github.com/your-username/helmet_app.git
cd helmet_app
flutter pub get
flutter run
```
